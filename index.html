About Me

I am a Computer Science graduate with a strong foundation in Natural Language Processing (NLP), Reinforcement Learning, Deep Learning, and Machine Learning. My academic journey has equipped me with both theoretical knowledge and practical skills in developing AI-driven solutions across various domains.

Currently, I am seeking opportunities for master's and PhD positions in Canada and the USA, where I can contribute to cutting-edge research in my areas of interest while further developing my expertise under the guidance of experienced researchers.

Selected Projects

Enhancing Biomedical Relation Extraction

Bachelor's Project

Developed a BERT-based NLP pipeline to detect Drug-Drug Interactions (DDIs) using the DDI Extraction 2013 dataset. The project focused on drug name entity recognition (DNER) and DDI classification, demonstrating the effectiveness of transformer models in biomedical relation extraction tasks.

NLP BERT Biomedical

LLM-Aware Search and Ranking

Built a semantic search pipeline integrating sentence embeddings and KNN-based ranking for LLM-enhanced applications. This project showcases my ability to work with modern NLP techniques and develop practical applications that leverage the power of large language models.

LLM Semantic Search Embeddings

Amazon Product Crawler and Indexer

Developed a Selenium-based web scraper to extract structured product data from Amazon. The extracted data was indexed and visualized using Elasticsearch and Kibana dashboards, demonstrating my skills in web scraping, data processing, and visualization.

Web Scraping Elasticsearch Data Visualization

Multilingual Search Engine

Implemented an information retrieval system for English and Persian datasets with customizable query processing and scoring. This project highlights my understanding of core information retrieval concepts and ability to work with multilingual data.

Information Retrieval TF-IDF Multilingual

Flower Classification

Developed image classifiers for flower datasets using Random Forest with K-means feature extraction and CNN-based transfer learning. The project involved comparing the performance of ensemble methods and fine-tuned deep models on small datasets.

Computer Vision Transfer Learning CNN

AI-Driven Othello Game

Developed an AI agent for the Othello board game using Minimax, alpha-beta pruning, dynamic depth selection, Beam Search, and Transposition Tables. This project demonstrates my understanding of search algorithms and game theory concepts.

Game AI Minimax Search Algorithms
